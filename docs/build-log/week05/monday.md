# Basic Electronic components
## Resistors
```
Resistors limit or control the flow of electric current in a circuit.
They are used for voltage division, current limiting, biasing, and setting reference levels.
Measured in ohms (Ω).
```
## Capacitors
```
Capacitors store electrical energy in an electric field.
They are used for filtering, smoothing power supply signals, timing circuits, and coupling/decoupling AC signals.
Measured in farads (F).
```
## Inductors
```
Inductors store energy in a magnetic field when current passes through them.
Commonly used in filters, power supplies, and RF circuits.
Measured in henrys (H).
```
## LED
```
LEDs are diodes that emit light when current flows through them in the forward direction.
They require current-limiting resistors to prevent burnout.
Used in indicators, displays, and lighting.
```
## Diodes
```
Diodes allow current to flow in one direction only.
They are used for rectification, protection, voltage regulation, and signal demodulation.
Types include rectifier diodes, Zener diodes, and Schottky diodes.
```
## Switches and Buttons
Switches control the connection or disconnection of electrical circuits.
```
SPST - Single pole single throw.
SPDT - Single pole double throw.
DPST - Double pole single throw.
DPDT - Double pole double throw.
```
## Transistors
Transistors act as electronic switches or signal amplifiers.

### BJT
```
Current-controlled device
Has Base, Collector, and Emitter
Good for amplification and switching
```
### MOSFET
```
Voltage-controlled device
Has Gate, Drain, and Source
Widely used in power electronics and microcontroller switching
```
### JFET
```
Voltage-controlled
High input impedance
Mostly used in analog circuits and amplifiers
```
### IGBT
```
Combines MOSFET input (voltage control) with BJT output (high current)
Used in high-power applications like motor drives and inverters
```
